# python-podgotovka-k-sobesedovaniy

Вот файл [MAIN](https://github.com/mezhibo/python-podgotovka-k-sobesedovaniy/blob/fef5660415d99d6a8d820004624dab37c6a95138/main.py)

Как это работает:

 - открывающие скобки кладём в стек;

 - при встрече закрывающей скобки проверяем, есть ли сверху соответствующая открывающая;

 - если нет совпадения или стек пуст, строка несбалансирована;

 - в конце стек должен быть пустым.

Примеры:

 - (((([{}])))) → Сбалансированно

 - {{[(])}} → Несбалансированно
